# TexView
## Overview
A python tool to automate the generation of a .tex file from an input folder containing several files to help with quick visualization of outputs. Currently supports .tex, .pdf, .png, .txt, .jpg, .jpeg, and .eps. 

## Description
Note that this package was developed using the conda distribution, adjust conda-related command to your own distribution. 

`TexView/` contains all the env files, they are automatically generated by conda. 

`main.py` is the python script. It contains a main function dealing with parsing and calling make_master() which actually generate the latex code. 

`input/` is the folder in which you place input files that will be displayed. 

## Use
Clone this repository onto your local device. Use `TexView.yml` file to load the appropriate dependencies using your regular python distribution and activate it. You are then set up to use the package.

Replace the files inside `input/` with the files you want to visualize -no harm was done to the frog currently living in the directory, it is only there to fill up the space. Then run the following commands to create the output latex file. 

```
python main.py -i input -o ouput.tex 
``` 

There are two options `--input [-i]` and `--output [-o]` the first is mandatory, while the second is main.tex by default but can be modified to your convenience.
